Err RSS
===============================

An ErrBot plugin for RSS feeds.

Dependencies
------------

 * Requests
 * Feedparser
 * Arrow

Features
--------

 * Watch feeds by url
 * Ignore watched feeds by title
 * List watched feeds

Usage
-----

    !rss watch <url>
    !rss list
    !rss ignore <title>
    !rss interval
    !rss interval 30
